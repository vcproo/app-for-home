# 做一个家庭记账的app
# 服务器
cloudflare 
该项目均使用cloudflare的免费层完成。
# 使用技术栈
Uni-app / HBuilderX（生态最成熟，适合国内开发者）
基于 Vue.js 的跨平台框架。

# 该项目是家庭内部使用的app,并且不会大面积推广，仅限内部使用。

# 功能说明
## 登录/注册
用户先注册，只能通过手机号注册，填写手机号密码即可，密码保存通过md5加密，密码无特殊限制，只要是6位以上即可。注册后登录，然后创建家庭，创建家庭之后会有一个邀请码，其他用户通过邀请码即可加入家庭。用户在后续的我的页面可以退出家庭。但他自己的账单保留，退出后只能查看自己的账单。

## 账单/统计
用户可以查看当前家庭的账单信息。用户可以记账。统计各个时间维度和支出收入维度的统计。

## 成员管理
查看当前家庭成员
